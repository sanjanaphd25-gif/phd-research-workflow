**Title:** Machine Learning in Medicine

**Full Citation:**
Rajkomar, A., Dean, J., & Kohane, I. (2019). Machine learning in medicine. *New England Journal of Medicine*, 380(14), 1347–1358. https://doi.org/10.1056/NEJMra1814259

**Summary:**
This paper discusses how machine learning is transforming healthcare by enabling predictive analytics, improving diagnostics, and supporting clinical decision-making.

**Key Points:**
- Machine learning models can process large-scale healthcare data
- Applications include imaging, EHR analysis, and disease prediction
- Challenges include data privacy and model interpretability
